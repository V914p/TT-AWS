### Objective of Worklog:

* Learn the overview of Amazon EC2 on AWS.
* Learn how to deploy and manage EC2 instances.
* Practice configuring Security Groups, Key Pairs, and SSH.
* Become familiar with Linux server administration workflow on the cloud.
* Learn basic EC2 monitoring and resource management.

### Tasks to be carried out this week:
| Day | Task | Start Date | Completion Date | Reference Material |
| --- | ---- | ---------- | --------------- | ------------------ |
| 2 | - Learn the overview of Amazon EC2 <br> - Study EC2 Instance Types <br> - Learn about AMI and EBS | 05/18/2026 | 05/18/2026 | AWS EC2 Documentation |
| 3 | - Create a Linux EC2 instance <br> - Configure a Key Pair <br> - Learn EC2 pricing basics | 05/19/2026 | 05/19/2026 | AWS EC2 Documentation |
| 4 | - Configure Security Group <br> - Open SSH Port 22 <br> - Manage Inbound and Outbound Rules | 05/20/2026 | 05/20/2026 | AWS VPC Documentation |
| 5 | - Connect to EC2 Linux via SSH <br> - Check the Linux server environment <br> - Practice basic Linux commands | 05/21/2026 | 05/21/2026 | AWS EC2 Documentation |
| 6 | - Monitor EC2 with CloudWatch <br> - Check CPU, Network, and Status Checks <br> - Manage instance state | 05/22/2026 | 05/22/2026 | Amazon CloudWatch Documentation |

### Week 5 Achievements:

* Understood the Amazon EC2 deployment workflow.
* Created and configured a Linux EC2 instance.
* Connected to the Linux server via SSH.
* Understood the role of Security Groups in controlling network access.
* Learned how Key Pair authentication works.
* Practiced basic Linux server management commands.
* Used CloudWatch to monitor EC2 metrics.
* Understood basic AWS Compute components:
  * EC2 Instance.
  * AMI.
  * EBS Volume.
  * Security Group.
  * Key Pair.
